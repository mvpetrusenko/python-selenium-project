Preconditions

1. on the local pc create a new folder with the name of your project

2. drag and drop it in VS code

3. install python from the official website

4. run VS code as an administrator

5. agree to install in extensions debugger for python: Python IntelliSense by Microsoft

6. in the terminal run: python --version to check if Python has been installed

7. create a file main.py in your project and run the code: print("Hello World!")

8. pip install pytest

9. pip install selenium

10. pip install webdriver-manager

11. pip install python-dotenv to load evnironment variable file: .env

12. pip install requests
